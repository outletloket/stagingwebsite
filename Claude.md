# Outlet Loket Claude Code Instructions

You are working on Outlet Loket, a Benelux-focused ecommerce outlet/deal store.

Live site:
https://outletloket.nl/

Staging site:
https://outletloketstaging.netlify.app/

Business goal:
Help scale revenue from €20K–€30K/month to €50K–€60K/month by improving product mapping, ecommerce SEO, conversion rate, mobile UX, checkout reliability, and production readiness.

Rules:
- Read the relevant code before changing anything.
- Keep changes small and reviewable.
- Prioritize revenue, SEO, conversion, speed, and reliability.
- Use natural Dutch for Dutch and Belgian customers.
- Keep future France/Germany expansion in mind, but focus on Benelux first.
- Never invent product data, prices, stock, brands, EANs, specs, or claims.
- Do not expose or change secrets, API keys, Stripe keys, webhook secrets, or payment data.
- Do not break WooCommerce cart, checkout, payment, stock, coupons, emails, or order flows.
- For Stripe/payment/order changes, include a clear test plan.
- For SEO work, consider titles, meta descriptions, H1s, canonicals, schema, crawlability, internal links, speed, and duplicate content.
- For every change, explain what changed, why it matters, and what to test.
- Be concise and use tokens efficiently.
